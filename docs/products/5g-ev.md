# 5G Enabled E-Vehicle

The 5G Enabled E-Vehicle platform by BaSig Wireless is an advanced research and prototyping system that integrates electric mobility with next-generation 5G communication technologies.

Designed for academic institutions, R&D labs, and industry innovators, this platform enables real-time experimentation in connected mobility, edge intelligence, and intelligent transportation systems.

---

## Key Features

- **5G Connectivity Integration**
  
  - Supports standalone (SA) architecture 
  - Integration with custom 5G core

- **V2X Communication Framework**
  - Vehicle-to-Vehicle (V2V)  
  - Vehicle-to-Infrastructure (V2I)  
  - Vehicle-to-Network (V2N)  
  - Vehicle-to-Pedestrian (V2P)

- **Edge Computing (MEC) Support**
  - Real-time data processing at the edge  
  - AI/ML inference for autonomous decision-making  

- **Onboard Sensor Integration**
  - GPS, IMU, cameras, LiDAR (optional)  
  - Real-time telemetry and analytics  

- **Remote Monitoring & Control**
  - Cloud dashboard integration  
  - Live vehicle diagnostics and control  

## Applications
- V2X communication
- Remote diagnostics
- Edge‑assisted mobility
- Smart transportation research
